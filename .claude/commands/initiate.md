# Initiate New Service Project

Create a new service project with the following structure:

- Service name: `$1`
- Service directory: `$1/`
- Planning document: `$1/PLAN.md`

## Tasks

1. Create the service directory at the repository root: `$1/`
2. Create a PLAN.md file inside the service directory with the following template:

```markdown
# $1

## Overview

[Brief description of the service]

## Architecture

[Architecture design and key components]

## Implementation Plan

### Phase 1: Foundation
- [ ] Project setup
- [ ] Basic structure

### Phase 2: Core Features
- [ ] [Feature 1]
- [ ] [Feature 2]

### Phase 3: Testing & Deployment
- [ ] Unit tests
- [ ] Integration tests
- [ ] Deployment configuration

## Technical Stack

- [List technologies and frameworks]

## Dependencies

- [List key dependencies]

## Notes

[Additional notes and considerations]
```

## Instructions

Please create the directory structure and PLAN.md file with the template above. Replace all instances of `$1` with the actual service name provided.

If the service name is not provided, ask the user to specify the service name.
